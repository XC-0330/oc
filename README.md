# OC角色图鉴

这是一个简洁、文艺风格的 OC 角色介绍网站模板，支持：
- 搜索角色信息与文章
- 首页展示随机古诗词
- 密码访问保护（默认密码：1231）

## 🧠 使用说明

1. 解压本项目
2. 上传所有文件到 GitHub 仓库（不要上传整个文件夹，只上传内容）
3. 打开仓库 Settings > Pages，启用 GitHub Pages：
   - Branch: `main`
   - Folder: `/ (root)`
4. 几十秒后，你的网站就能通过地址访问了：

   ```
   https://你的用户名.github.io/你的仓库名/
   ```

🎉 Done！

## 💬 默认密码

```txt
1231
```

可在 `js/script.js` 文件中修改。
